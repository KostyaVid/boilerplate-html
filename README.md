# boilerplate-html

Template based [parcel.js](https://parceljs.org)

## Futures

-   Add inline Template html
-   Add normilize.css
-   Add Animate button Burger
-   Add adaptive menu
-   Add True focus in the mobile menu
-   Add matchMedia triggers
-   Use [parcel.js](https://parceljs.org)

## Install

```
yarn install
```

## Work

```
yarn start
```

## Build

```
yarn build
```
